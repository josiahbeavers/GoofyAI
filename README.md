# GoofyAI Beta 1.0

GoofyAI is a local AI chat app built with Next.js and Ollama.

It has:
- Local browser accounts
- Saved chat history
- Streaming responses
- Custom themes
- A compact ChatGPT-style sidebar
- A goofy but useful assistant personality

## Important

GoofyAI accounts are saved only in your browser. They are not real online accounts.

GoofyAI uses Ollama locally, so each person must run Ollama on their own computer.

## Requirements

Before running GoofyAI, install:

- Node.js
- npm
- Ollama

You also need the llama3 model:

```bash
ollama pull llama3
